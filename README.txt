INSTALLATION

FIXME is normally installed using the easy_install script from
the Python setuptools package.  The setuptools library is provided
by Linux distributions as "python-setuptools" package.  It can be
also obtained from http://pypi.python.org/pypi/setuptools .
To install the FIXME package, use

    easy_install -f http://www.diffpy.org/packages/ diffpy.fixme

To install diffpy.fixme to a specific directory DIR, make sure it is
included in the PYTHONPATH environment variable and execute

    easy_install -d DIR -f http://www.diffpy.org/packages/ diffpy.fixme

For more options, see "easy_install --help" and setuptools documentation
at http://peak.telecommunity.com/DevCenter/setuptools
